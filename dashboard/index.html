    📖 如何使用自动监控系统

    1 安装依赖: pip install playwright 然后 playwright install chromium
    2 运行监控脚本: python monitor.py（爬取实时数据）或 python monitor.py --demo（生成演示数据）
    3 运行分析引擎: python analyzer.py（计算蓝海评分和告警）
    4 打开本仪表盘，点击「从文件加载」导入 dashboard/data.json
    5 设置Windows定时任务: 每天北京时间06:00自动运行 monitor.py + analyzer.py
